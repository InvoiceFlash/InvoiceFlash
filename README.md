# InvoiceFlash 

InvoiceFlash is a self-hosted open source application for managing your invoices, clients and payments.

### Requirements
To use InvoiceFlash application you should have already installed:

*   **_PHP_** >=5.0 
*   **_MySQL_** >=4.1 server
